# Fermat-Discrete-Lattice-Proof

### 概要 / Abstract
フェルマーの最終定理を、抽象数論としてではなく、三次元離散格子における「剛体シェルの再構成」という物理的制約から証明した論文です。

This paper provides a physical proof of Fermat's Last Theorem, reinterpreting it not as abstract number theory, but as a geometric constraint on "rigid shell reconstruction" within 3D discrete lattices.

### 核心の論理 / Core Logic
- **原本仕様（163）**: 宇宙の最小単位を離散格子として定義。
- **構造的幾何学排他律**: $Z^3$ から $(Z-1)^3$ を引き抜いた際に残る「1ユニットの頂点属性」が、新しい立方体への再構成を物理的に拒絶する。
- **排他律の剛性**: 数学的近似を排した、離散空間における決定論的な証明。
